# Alliances-Trahisons

Projet réalisé dans le cadre du Parcours IHM de Décembre 2017 à Mars 2018, sur table tactile et appareil Android.

Alliances & Trahisons est un jeu sur smartphone et table tactile. 
Accès sur la compétition à travers la conquête de planètes par le déplacement de vaisseaux, 
les utilisateurs peuvent aussi collaborer en échangeant des ressources et en fondant des alliances.

Vous trouverez la méthode de déploiement dans le fichier "Déploiement.pdf" 
et la description du projet dans le fichier "Spécifications.pdf".
